One-time Firebase Setup

Takes 5 minutes. After this your progress syncs automatically across every device — no keys, no copying, just sign in with Google.

Steps

1

Go to console.firebase.google.com and sign in with your Google account

2

Click Add project → name it anything → disable Google Analytics → click Create

3

In the left sidebar click Build → Firestore Database → Create database → Start in test mode → pick any location → Enable

4

In left sidebar click Authentication → Get started → Google → Enable → Save

5

Click the gear icon (Project Settings) → scroll down to "Your apps" → click the </> web icon → register app (any name) → copy the firebaseConfig object

6

Paste the config object below and click Save
